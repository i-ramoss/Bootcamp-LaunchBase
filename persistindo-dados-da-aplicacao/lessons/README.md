## :pushpin: Sobre:

Foi desenvolvido toda a estruturação de um banco de dados para a aplicação Gym Manager, não mais salvando os dados em um arquivo json, mas sim em um banco de dados relacional, no caso o PostgreSQL. <br>
Além disso criou-se toda uma lógica de paginação para os alunos e professores cadastrados, tanto no frontend com a DOM, quanto no backend nos controllers.
